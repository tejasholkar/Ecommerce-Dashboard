# Ecommerce-Dashboard
Interactive Ecommerce Sales Dashboard built using Power BI with DAX, Power Query, and data modeling to analyze sales, profit, customers, products, and payment trends.

# 📊 Ecommerce Sales Dashboard | Power BI

An interactive **Power BI dashboard** developed to analyze ecommerce sales data and provide meaningful business insights through dynamic visualizations and KPIs.

---

## 📌 Project Overview

This project focuses on transforming raw ecommerce sales data into an interactive dashboard that helps monitor business performance. The dashboard enables users to explore sales trends, profit distribution, customer behavior, and regional performance using filters and interactive visuals.

---

## 🎯 Objectives

- Analyze overall sales and profit performance.
- Identify top-performing product categories and sub-categories.
- Compare sales across different states.
- Analyze customer purchasing behavior.
- Understand payment mode preferences.
- Build an interactive dashboard for business decision-making.

---

## 📊 Dashboard Features

- 📈 Sales & Profit KPIs
- 🛍️ Category & Sub-Category Analysis
- 🌍 State-wise Sales Analysis
- 👥 Customer Insights
- 💳 Payment Mode Distribution
- 🎛️ Interactive Filters & Slicers
- 📅 Time-based Sales Analysis
- 📌 Clean and User-Friendly Dashboard Design

## 📂 Dataset

The project uses two datasets:

### Orders.csv
Contains order information including:
- Order ID
- Order Date
- Customer Name
- State
- City

### Details.csv
Contains transactional details including:
- Order ID
- Amount
- Profit
- Quantity
- Category
- Sub-Category
- Payment Mode

The datasets are connected using **Order ID** to build the data model.

---

## 📌 Dashboard Insights

The dashboard helps answer questions such as:

- Which product categories generate the highest sales?
- Which states contribute the most revenue?
- Which payment modes are most frequently used?
- Which products generate the highest profit?
- How do sales and profit vary over time?
- Which customers place the highest-value orders?
